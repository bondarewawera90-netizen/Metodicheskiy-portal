4. КЛЮЧЕВЫЕ ЗАДАНИЯ

📋 Копировать задание 1

Задание 1. Группировка языков
Распределите примеры: русский, немецкий, химические обозначения, азбука Морзе, формулы, Python, китайский, французский → естественные / формальные языки.

📋 Копировать задание 2

Задание 2. Вставьте пропущенные слова
Каждый язык строится из ______. Совокупность знаков — ______. Характеристика алфавита — ______. Языки, возникшие исторически — ______; созданные по правилам — ______.

📋 Копировать задание 3

Задание 3. Перевод в формальный язык
«Какое число при умножении на 2 даёт 10?» → записать уравнением. «К числу 5 прибавили неизвестное число и получили 12» → записать уравнением.

5. ДИАГНОСТИКА И РЕФЛЕКСИЯ

Диагностика: мини-тест: определить естественные и формальные языки; объяснить понятие мощности алфавита.
Рефлексия: карточка 3-2-1: 3 понятия, 2 задания, 1 вопрос.
Домашнее задание: таблица «Термин — определение — пример» для понятий: символ, алфавит, мощность, естественный язык, формальный язык, двоичный алфавит.
